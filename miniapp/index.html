Maybe OSINT — mini app

Концепт

Maybe — это лёгкий и полностью бесплатный бот для поиска по никнеймам.

* Сейчас доступен поиск только по Maigret

Основной функционал:

  • Поиск никнейма — длина от 2 до 30 символов
  • Отчёт в PDF либо HTML — удобно сохранять и делиться
  • История поиска — сохраняются первые 100 и последние 100 запросов
  • Доступ к результатам по ссылке — формат https://t.me/---?start=result_b7d38067
  • Система рангов — больше 100 «уровней»
  • Топ-10 пользователей — с возможностью скрыться из рейтинга

Планы по развитию:

  • Доработка существующего функционала
  • Создание собственного клиента в двух вариантах:
    • Сайт
    • Мобильное приложение
